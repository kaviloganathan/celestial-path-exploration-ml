# Celestial Path Exploration Project

## Description
This project focuses on trajectory analysis of celestial bodies using Python and Django.  
It integrates AI models for prediction and uses energy data processing and grid modeling for accurate results.

## Technologies Used
- Python
- Django
- MySQL
- HTML, CSS
- JavaScript

## Features
- Energy data processing
- Grid structure modeling
- AI-based trajectory prediction
- Client, Power Source, Grid Structure, System Maintenance, Admin modules
